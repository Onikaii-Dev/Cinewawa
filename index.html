🎬 Importer depuis Letterboxd

  1. Sur letterboxd.com (depuis un navigateur) : ton avatar → Settings → onglet Data.
  2. Clique « Export your data » → un fichier ZIP se télécharge.
  3. Dézippe-le (clic droit → Extraire tout).
  4. Dedans, prends ratings.csv (tes films notés) ou watched.csv (tout ce que tu as vu).
Les notes /5 sont converties en /10 (4½ ★ → 9), les dates de visionnage sont reprises, et les films déjà présents sont ignorés. Les affiches se récupèrent ensuite fiche par fiche avec 🔍.

☁️ Synchro cloud (GitHub)

Ta sauvegarde est stockée sur ton compte GitHub. Colle le même token sur chaque appareil (PC, téléphone…) et tout se synchronise automatiquement.
❓ Première fois ? Le tuto complet (2 min)
  1. Crée un compte gratuit sur github.com (si tu n'en as pas).
  2. Va sur github.com/settings/tokens → « Generate new token (classic) ».
  3. Donne un nom (ex : cinewawa), expiration « No expiration », et coche UNIQUEMENT la case « gist ».
  4. Clique « Generate token » et copie le code ghp_… (montré une seule fois : garde-le dans tes notes).
  5. Colle-le ci-dessous, choisis un pseudo, puis « Activer » — ta sauvegarde cloud est créée.
  6. Sur ton téléphone : ouvre ce même site → ☁️ → colle le même token → Activer. Tout se retrouve !
Créé sur github.com → Settings → Developer settings → Tokens (classic) → coche uniquement « gist ». Il reste sur cet appareil, il n'est envoyé qu'à GitHub.
